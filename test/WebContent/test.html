<>로 표현하는 태그가 있다.
여는 태그와 닫는 태그가 있고, 여는 태그만 사용하는 것도 있다.

ip 주소 cmd에서 확인하는 방법 : ipconfig

로컬 ip주소 : 실제 ip 주소, localhost, 127.0.0.1

language에 해당하는 것

  1. html : 웹문서를 만드는 언어로,웹문서의 기본 뼈대부분이다.
  2. css(cascading style sheet) : 웹문서를 보기좋게 꾸미는 디자인표현(형용사,부사)
  3. javascript/jQuery : 프로그래밍적 요소로 사용자의 행위에 의한 동작(동사)

요소(element) html = 부모요소 -> head,body= (자식이며, 서로 형제요소이다.) 데이터 확인 : 콘솔(console)

1. 블록 레벨 텍스트 태그

  1. h(heading) 태그 : h1 이 가장 크고 h6이 가장 작은 것이며, 혼자서 한 줄을 꽉 차게 표현하는 태그이다.
  2. p(paragraph) 태그 : 단락을 표현하는 태그로 h와 div에 비해 margin 영역이 있어 너비가 넓다.
  3. hr(horizontal) 태그 : 주제가 바뀌거나 분위기 전한 할 때 가로선을 긋는 것이다.
  4. br 태그 : 띄어쓰기 할 때 사용한다.
  5. div(division) 태그 : 영역을 나눌 때 사용한다.

주석처리를 하고싶으면 <> 안에 !-- 를 넣으면 된다.
하지만 이것을 다 쓰기 힘들기 때문에 ctrl + shift + (c & /) 로 키보드 사용하면 된다.
한 줄만 주석처리 한 것은 취소를 할 수 있지만 여러줄 주석처리하면 취소할 수 없다.


위에 쓴 내용을 블럭으로 처리하고 싶으면 : shift + 방향키
위에 쓴 내용을 복붙하려면 : ctrl + alt + 방향키
코드라인 블럭이동 : alt + 방향키
한 블럭 없에기 : ctrl + d
자동완성 기능 : ctrl + spacebar

2. 인라인 레벨 택스트 태그

  1. strong 태그 : 글씨를 굵게 표시하는 태그
  2. span 태그 : 텍스트 일부만 특정 스타일 처리를 할 때 사용한다.
  3. 목록 만드는 태그 : 순서를 없이 하는(unordered list) ul (예로는 li가 있다.)과, 순서 있게 하는(ordered list) ol( li가 있다.)로 나뉜다.

    ul의 list-style-type 이 가장 기본 bisc이다.

    ol의 list-style-type 이 가장 기본 decimal(정수)이다.

    숫자, 알파벳, 로마숫자로 바꿀 수 있는데
    숫자 시작값 지정 : start=''
    알파벳 대소문자, 로마숫자 지정 : type=''


    3. 표를 만드는 태그

    : 행(세로)과 열(가로)로 이루어진 표의 형태를 만들 때 사용하는 태그로, table 태그라고 한다.
    • tr(table row) 태그 : 행에 해당하는 셀 태그
    • td(table data) 태그 : 행 내부 열에 해당하는 셀 태그
    • th(table head) 태그 : 행 내부의 제목에 해당하는 셀 태그

    caption태그 : 표에 사용하는 표제목 표시 태그
