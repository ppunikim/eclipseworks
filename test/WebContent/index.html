1. 블록 레벨 텍스트 태그 :


혼자서 한 줄을 차지하고 너비가 100%꽉 차게 표시되는 태그

1) h태그 : heading태그 - 제목을 표시하는 태그로 h1~h6까지 있음

가장 큰 제목태그

가장 작은 제목태그

2) p태그 : paragraph - 단락을 표시하는 태그로 줄바꿈 없이 텍스트가 한 줄로 표시됨
같은 속성이면 같은 라인에 표시하고, 아니라면 줄바꿈을 한다.

3) br태그 : 줄바꿈을 처리하는 태그로 닫는 태그가 없이 쓴다.

4) hr태그(horizontal) : 주제가 바뀌거나 할 때 분위기 전환하는 가로선 표시로 닫는 태그가 없다.

5) div태그(division): 영역을 나누는 블록 태그
>

2. 인라인 레벨 텍스트 태그

한 줄을 차지하지 않는 태그로 화면에 표시되는 컨텐츠만큼의 너비를 차지한다.

실행에서 제외하고자 하는 코드는 주석처리를 하는 것으로, 이렇게 생김
이것의 단축키는 한줄이면 ctrl + shift + c 누르면 되고(해제 가능),
여러줄이면 ctrl + shift + / 이다.(해제 불가능)
이 때, 줄에 한부분이라도 태그가 돼 있으면 처음부터 끝까지 주석처리 된다.

1) strong태그 : 텍스트를 굵게 표시하는 인라인 레벨 태그

굵은 텍스트 자동완성 기능 : ctrl + spacebar
위에 쓴 내용 블럭 설정 : shift + 방향키
위에 썼던 내용을 복붙하는 방법 : ctrl + alt + 방향키
코드 라인 블럭 이동 : alt + 방향키
한 줄씩 없에는 것 : ctrl + d

2) span 태그 : 특별한 뜻은 없지만 텍스트의 일부만 스타일 적용을 하고자 할 때 주로 사용
