  • 메뉴 alert창 띄우기

    • 링크

    • a태그 : 링크는 현재 웹 페이지에서 다른 페이지로 연결하기 위한 형태
      -> target='_blank' 이것은 따로 창이 또 뜨도록 만드는 것(창2개)
      -> target='_self' 이것은 내가 지금 있는 창으로 창을 여는 것/ 굳이 지정 안해도 된다.(창1개)
    • target : 현재 페이지에서 특정 위치로 연결할 수 있음
      -> <>안에 " a href = '#속성이름' 하고 연결할부분명 적어준 후에 <>닫아주면 된다.
      -> id값은 페이지에 단 한개 있어야 한다.
      -> #이 있는 곳의 글씨를 누르면 id값에 동일한 이름 지정해준 곳으로 위치를 이동한다.
    • text-decoration: none; 란, 링크에 있는 밑줄 없에준다.
      color: initial; 란, 클릭을 해도 색이 변하지 않도록 한다.
      -> color: interit; 란, 처음 색상으로 지정하는데, 색이 변하지 않도록 하는 것이다.
      cursor: pointer; 란 커서를 화살표 모양이 아닌 손가락 모양으로 지정하는 것이다.
    • (경고)안내창 실행 : <>에 a target= '_blank' onclick='alert("메뉴 클릭")'>메뉴 입력
      -> alert 바깥쪽과 ()안쪽의 따옴표 갯수가 틀려야 한다.

    • 현재 페이지가 있는 폴더가 아닌 폴더에 있는 이미지 파일을 찾아가는 경우 " . " : 현재위치," .. ": 상위폴더로 나간 위치
    • img 태그 : 이미지 태그로, 폴더 내의 이미지 표현과 웹상의 이미지 표현하며, 이미지태그는 여는 태그만 있다.
    • alt : 이미지파일이 정상로딩이 되지 않는 경우 이미지에 대한 설명을 넣는 속성
    • title : 마우스를 가져다 대었을 때 보이는 설명을 말풍선으로 넣는 속성
