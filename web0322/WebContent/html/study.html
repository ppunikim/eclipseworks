지난 수업내용

    form태그 : 사용자가 입력받는 화면을 구성할 때 사용하는 태그로 입력한 데이터를 다른페이지에 보낼수 있는 태그

  • 입력태그:input type="" name=""이 기본구조
    • text한줄 택스트 입력
    password입력 회로가 보이지 않는 비밀번호 태그
    hidden실제 소스가 화면에서는 보이지 않는 히든 태그
    file첨부할 파일을 선택하는 태그
    radio여러 항목중 한 가지만 선택하는 태그
    checkbox여러 항목중 여러개 선택 가능한 태그
    submit폼 내 입력데이터를 전송하는 태그

  • textarea태그: textarea name="" 이 기본 구조 여러줄을 입력하는데 사용하는 입력 텍스트 태그

  • select태그 : select name="" 이 기본 구조 여러개의 목록에서 선택하는 드랍다운 목록태그
